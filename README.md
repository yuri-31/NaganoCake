# ながのCAKE

## サイト概要

### サイトテーマ
ながのCAKE(仮想の洋菓子店)のネット販売・注文情報管理ウェブサイトです。

### ターゲットユーザ
- 当洋菓子店のオンライン販売を利用したい人
- 当洋菓子店の従業員でサイトの情報更新や顧客情報の管理をしたい人

### 主な利用シーン
- 注文者がネット上で商品を閲覧したいとき
- 注文者がネット上で注文した商品の製作状況・発送状況を確認したいとき
- 洋菓子店の従業員が顧客情報を参照したいとき
